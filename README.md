# EventCentral

EventCentral is an event discovery and hosting platform where anyone can find events near them, RSVP to attend, leave reviews, and host their own events under an organization profile. The platform is built for both attendees looking to explore what is happening locally and organizers who want a dedicated space to publish and manage their events.

---

## Features

### Event Discovery

Browse all public events from the Discover page. Filter results by keyword, category, date range, or location. Each event card shows the title, date, location, category, and host organization. Pagination keeps results manageable as the catalog grows.

### Map View

An interactive map interface powered by Google Maps displays events as markers across a geographic area. A sidebar lets you filter by keyword, location, date range, event category, and search radius. Clicking a marker highlights the corresponding event in the list. The map and list stay in sync as filters change.

### Event Pages

Each event has a dedicated detail page with the full description, date and time, location with an embedded map, host organization, capacity, and category. Logged-in users can RSVP directly from the page. The attendee count and a sampling of attendee avatars are displayed in real time.

### RSVP

Authenticated users can confirm attendance at any event with a single click. The RSVP is reflected immediately on the event page. Users can view their upcoming and past RSVPs from their profile. A full attendee list is available in a dialog on the event page.

### Reviews and Ratings

After attending an event, users can leave a star rating (1 to 5) and a written review. Reviews are displayed on the event page with the reviewer's username and avatar. Users can edit their own reviews at any time. An AI-powered summary, generated by Claude, synthesizes the key themes from all reviews into a concise paragraph shown above the review list.

### Organizations

Users can create one or more organization profiles to host events under a named brand. Each organization has a name, avatar, banner image, description, contact email, phone number, and website link. The organization page lists all events the organization has published, separated into upcoming, past, and cancelled tabs with pagination. A founder attribution section links to the owner's profile.

### Organization Management

Organization owners can edit every field of their organization profile, including uploading or replacing the avatar and banner image. Events can be created, edited, and deleted from within the organization page. Organizations can be permanently deleted, which removes all associated data.

### Event Creation and Editing

Authenticated users can create events from within any organization they own. The creation form accepts a title, description, start and end date and time, location (with address autocomplete and geocoding), capacity, category, and a cover image. Events can be edited or cancelled after creation. Cancellation marks the event without deleting it so attendees are informed.

### Analytics Dashboard

Organization owners have access to a private analytics dashboard. The dashboard covers three areas:

**Engagement**: Total RSVPs, a timeline chart of RSVP activity over 7 days, 30 days, or all time, and a ranked list of top events by attendance.

**Reviews**: Average star rating, total review count, rating distribution chart, and a feed of recent reviews with reviewer details.

**Reach**: Total organization profile views, total event page views, and a per-event view breakdown.

All sections respect the selected time range filter.

### User Profiles

Every registered user has a profile page showing their username, avatar, bio, and phone number. The profile page lists upcoming and past RSVPs alongside any organizations the user owns. Users can edit their display name, avatar, phone number, and bio at any time. Accounts can be permanently deleted from the profile page.

### Authentication

Users can sign up and log in with an email address and password or with a Google account via OAuth. New accounts are guided through a short onboarding step to set a username, phone number, and bio before reaching the main application.

### Landing Page

The public landing page introduces the platform to new visitors. It features a hero section with sign-up and browse calls to action, a category showcase with live event counts, a how-it-works walkthrough, a featured events carousel, and a section encouraging users to start hosting.

### Theming

The application supports both light and dark modes. The active theme follows the user's system preference by default and can be toggled manually from the header.
